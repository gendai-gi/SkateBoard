# コナン君ボードプロジェクト   

学校で台車に乗って押して遊んでたら「なんか押すのめんどくねｗ」ってなって作ろうと思ったのがコナン君ボード。   
まあただスケボーにモータつけるだけなんだけどねｗ 超音波センサーなんかつけたけど結局モーター選びと電源どうするかってのであぼーんした
